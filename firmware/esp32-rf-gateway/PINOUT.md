# ESP32 RF Gateway — Pinout

ESP32 + CC1101 433 МГц приёмник для беспроводных датчиков (Ea2 и совместимые).

## CC1101 (SPI VSPI)

| CC1101 | ESP32 | Назначение                                   |
|--------|-------|----------------------------------------------|
| VCC    | 3V3   | Питание (только 3.3 В!)                      |
| GND    | GND   |                                              |
| SCK    | 18    | SPI clock                                    |
| MISO   | 19    | SPI MISO (= GDO1 на CC1101)                  |
| MOSI   | 23    | SPI MOSI                                     |
| CSN    | 5     | Chip select                                  |
| GDO0   | 4     | Async data out (демодулированный битстрим)   |
| GDO2   | 2     | (опционально) RSSI/CCA, не используется      |

CC1101 настроен на 433.92 МГц, OOK-модуляция, async serial out — GDO0 выдаёт raw сигнал, ESP32 ловит фронты по прерыванию и собирает пульсы.

## Прочее

| Пин | Назначение                      |
|-----|---------------------------------|
| 0   | BOOT — кнопка сброса (PULLUP)   |
| 2   | Built-in LED + GDO2 (свободен)  |

## Питание

CC1101 чувствителен к питанию — рекомендуется поставить керамику 100nF + электролит 10мкФ как можно ближе к VCC модуля. Антенна — четвертьволновый штырь 17.3 см или спираль.
