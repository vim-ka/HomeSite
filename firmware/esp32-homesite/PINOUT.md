# ESP32 Pinout — HomeSite Boiler Unit

## Датчики

| GPIO | Функция | Подключение | Примечание |
|------|---------|-------------|------------|
| 4 | OneWire (DS18B20) | DATA → GPIO4, подтяжка 4.7kΩ к 3.3V | Все температурные датчики на одной шине |
| 15 | DHT22 | DATA → GPIO15, подтяжка 10kΩ к 3.3V | Температура + влажность (опционально) |
| 34 | ADC — давление отопления | Выход датчика через делитель напряжения | ADC1, input-only, по умолчанию |
| 35 | ADC — давление водоснабжения | Выход датчика через делитель напряжения | ADC1, input-only, по умолчанию |

### Делитель напряжения для датчиков давления

Датчик 0.5–4.5V → делитель (4.7kΩ + 3.3kΩ) → 0.21–1.85V на вход ESP32 (макс 3.3V).

```
Датчик OUT ──┬── 4.7kΩ ── GND
             │
             └── 3.3kΩ ── GPIO 34/35
```

## Системные

| GPIO | Функция | Примечание |
|------|---------|------------|
| 0 | Кнопка сброса | BOOT кнопка на плате. Удержание 5с = сброс конфигурации |
| 2 | Встроенный LED | Горит = рабочий режим, мигает = чтение датчиков |

## Свободные пины

Следующие GPIO не используются и доступны для расширения:

| GPIO | Примечание |
|------|------------|
| 36 (VP) | ADC1, input-only |
| 39 (VN) | ADC1, input-only |
| 15 | Занят DHT22, свободен если DHT не используется |

## Ограничения ESP32

- **GPIO 34, 35, 36, 39** — только вход (input-only), нет внутренней подтяжки
- **GPIO 6–11** — заняты SPI flash, не использовать
- **GPIO 12** — влияет на напряжение flash при загрузке (bootstrap pin), осторожно при отладке
- **ADC2 (GPIO 0, 2, 4, 12–15, 25–27)** — не работает при активном WiFi, поэтому давление на ADC1 (34, 35)
